## Instruction
___
–êll commands are executed from the root directory of the project:

1. Install project:
```
    make install
```
2. Start the game:
```
    brain-{name}
```

## List of games

* brain-even
* brain-calc
* brain-gcd
* brain-prime
* brain-progression
